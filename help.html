Basic Operations

  1. Make sure you have a DuckDuckGo (DDG) email account (go to https://duckduckgo.com/email to check)
  2. Go to https://hamster45105.github.io/DuckAddressSend/
  3. Input the email address that you want to send an email TO.
  4. Input either a private or your personal duck address address that you want to send your email FROM.
  5. Click "Create Address!"
  6. Your address will be displayed underneath button.
  7. Now, just send the email from your own email address (eg. @gmail.com) to that special created email address.

Functions

Copy to Clipboard

This copies the created address to your clipboard. An alert will be displayed to confirm that it worked

Errors

Please enter a valid duck address: All valid duck addresses end in @duck.com, so please do one that ends in that.

Please enter a valid email address: Pretty self-explantory, just enter a proper address

Invalid DuckDuckGo API Key!: This means that the API key that you inputted is not valid. Double check to make sure that you have put the right one in. They are case-sensitive.
